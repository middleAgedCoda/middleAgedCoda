# SHAMWARI Server (Lean)

Fastify + Prisma API skeleton.

## Commands
- Install: `npm i`
- Dev: `npm run dev`
- Build: `npm run build && npm run start`
- Prisma: `npm run prisma:migrate` / `npm run prisma:generate`

## Env
Copy `.env.example` to `.env` and adjust.

## Endpoints
- `GET /health`
- `POST /quotes` → placeholder quote calc
- `POST /transfers` → create transfer intent
